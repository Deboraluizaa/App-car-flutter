Controle de Quilometragem

Sobre o Projeto

Este sistema foi desenvolvido para registrar e monitorar a quilometragem de veículos. Com ele, é possível adicionar registros com fotos e visualizar o histórico de forma prática e intuitiva.

Funcionalidades

  • Adicionar registros de quilometragem com fotos.
  • Visualizar o histórico de registros.
  • Persistência local dos dados usando SQLite.
  • API RESTful para manipulação de dados.

Tecnologias Utilizadas

  • Frontend: Flutter
  • Backend: ASP.NET Core
  • Banco de Dados: SQLite
